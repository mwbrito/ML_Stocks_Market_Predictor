# Machine Learning Engineer Nanodegree
## Projeto final

### Instalação

Este projeto requer **Python 2.7** e a instalação das sequintes bibliotecas Python:

- [NumPy](http://www.numpy.org/)
- [Pandas](http://pandas.pydata.org)
- [matplotlib](http://matplotlib.org/)
- [scikit-learn](http://scikit-learn.org/stable/)

Você também vai precisar ter instalado o software [Jupyter Notebook](http://ipython.org/notebook.html)

### Execução

No terminal ou janela de comando, navegue até o diretório raiz do projeto `stock_market_predictor/` (que contém este README) e execute um dos comandos abaixo:

```bash
ipython notebook capstone-stock_market_predictor.ipynb
```  
ou
```bash
jupyter notebook capstone-stock_market_predictor.ipynb
```

O arquivo será aberto no Jupyter Notebook.

## Dados

Nestes projetos são utilizados dados de negociações de ações, obtidos no site [Yahoo Finance](https://finance.yahoo.com/).Os arquivos podem ser encontrados nos diretórios `stock_market_predictor/files` e `stock_market_predictor/files_v2`.
